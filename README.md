# Beardude Land (preact ver.)

## Development Workflow

**Start a live-reload development server:**

```sh
npm run dev
```

**Testing with `mocha`, `karma`, `chai`, `sinon` via `phantomjs`:**

```sh
npm test
```

**Generate a production build in `./build`:**

```sh
npm run build
```

**Start local production server:**

```sh
npm start
```
